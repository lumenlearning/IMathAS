formpoly3fromstring

formpoly3fromstring(variable, polynomialstring, [IsFraction=true])

Creates an array of coefficients whose position in the array corresponds to the power of the variable. Each element is stored in the form of an array(numerator, denominator).

  • variable: a string that contains the variable.
  • polynomialstring: a string that contains the polynomial.
  • IsFraction: optional - a boolean that stores the coefficients as fraction arrays.
Examples
$dividend = formpoly3fromstring("x", "3x^4+2x^2+1", false)
// array(1,0,2,0,3)

formpoly3fromresults

formpoly3fromresults(results, [IsFraction=true])

Creates an array of coefficients whose position in the array corresponds to the power of the variable. Each element is stored in the form of an array(numerator, denominator).

  • results: from dividepoly3, $results[*][1]
  • IsFraction: optional - a boolean that stores the coefficients as fractions.
Examples
$quotient = formpoly3fromresults($results[0][1])

writepoly3

writepoly3(poly3, [variable="x", IsFraction=true, HideZero=true])

Creates a string that represents the polynomial. Each element is stored in the form of an array(numerator, denominator).

  • poly3: an array of coefficients to build the polynomial.
  • variable: optional - a string that contains the variable.
  • IsFraction: optional - true if the elements of poly3 are fraction arrays. Set false if values are integers or decimals.
  • HideZero: optional - a boolean that stores a flag to hide zero coefficients.
Examples
$dividend = writepoly3(array("1,0,2,0,3"), "x", false)
// "3x^4+2x^2+1"

dividepoly3

dividepoly3(dividend, divisor, [IsFraction=true])

Does the polynomial long division and returns an array of results.

  • dividend: a poly3-formed dividend.
  • divisor: a poly3-formed divisor.
  • IsFraction: optional - a boolean that stores the coefficients as fractions.

$result will contain an array in the following form:

  • $result[0][0] = $dividend
  • $result[0][1] = the power, array(numerator, denominator)
  • $result[0][2] = a poly3 polynomial by multiplying $result[0][1] and the divisor
  • $result[0][3] = the poly3 polynomial that needs to be subtracted (-$result[0][2])
  • $result[1][0] = the resultant dividend (at least 1 power smaller than the previous)
  • etc.
Examples
$result = dividepoly3($dividend, $divisor)

longdivisionpoly3

longdivisionpoly3(dividend, divisor, variable="x", [displayASCIIticks=true, IsFraction=true, HideZero=true, ShowAsSubtraction=false])

Does the long division and returns a string of the results.

  • dividend: a poly3-formed dividend.
  • divisor: a poly3-formed divisor.
  • variable: a string that contains the variable.
  • displayASCIIticks: optional - a boolean flag to put tick marks around each monomial.
  • IsFraction: optional - a boolean that stores the coefficients as fractions.
  • HideZero: optional - a boolean that stores a flag to hide zero coefficients.
  • ShowAsSubtraction: optional - a boolean that when set to true shows the subtraction.
Examples
$result = longdivisionpoly3($dividend, $divisor)
// $result will contain a string of HTML table code 
// that can be displayed in the answer.
