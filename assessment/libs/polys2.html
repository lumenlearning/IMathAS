formpoly2

formpoly2(coefficients, powers, powers2)

Creates a polynomial object with two variables. Use writepoly2 to create a display form of the polynomial.

  • coefficients: list/array of coefficients.
  • powers: list/array of powers for the first variable.
  • powers2: list/array of powers for the second variable.
Examples
// 2x^2 + 3xy + 4y^2
$p = formpoly2(array(2,3,4), array(2,1,0), array(0,1,2))

writepoly2

writepoly2(poly, [var, var2, showzeros])

Creates a display form for a polynomial object.

  • poly: polynomial object, created with formpoly2.
  • var: first variable, defaults to x.
  • var2: second variable, defaults to y.
  • showzeros: optional, defaults to false. If true, shows zero coefficients.
Examples
$out = writepoly2($p)  // "2x^2+3xy+4y^2"
$out = writepoly2($p, "s", "t")  // uses s and t as variables

addpolys2

addpolys2(poly1, poly2)

Adds polynomials, arranging terms from highest to lowest powers.

Examples
$sum = addpolys2($p1, $p2)

subtpolys2

subtpolys2(poly1, poly2)

Subtracts polynomials: poly1 - poly2, arranging terms from highest to lowest powers.

Examples
$diff = subtpolys2($p1, $p2)

multpolys2

multpolys2(poly1, poly2)

Multiplies polynomials.

Examples
$prod = multpolys2($p1, $p2)

scalepoly2

scalepoly2(poly, c)

Multiplies each term of poly by the constant c.

Examples
$scaled = scalepoly2($p, 3)

polypower2

polypower2(poly, power)

Calculates poly^power.

Examples
$squared = polypower2($p, 2)

getcoef2

getcoef2(poly, degree, degree2)

Gets the coefficient corresponding to the degrees specified. If no such term is defined, 0 is returned (since that is the coefficient!).

  • poly: polynomial object, created with formpoly2.
  • degree, degree2: degree of the first and second variable of the term to get the coefficient of.
Examples
$coef = getcoef2($p, 2, 1)  // coefficient of the x^2*y term
