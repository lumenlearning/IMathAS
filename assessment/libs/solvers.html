bisectionsolve

bisectionsolve(function, xmin, xmax)

Applies bisection to find a root of the function on the interval [xmin, xmax]. This function only works if there is exactly one root in the interval.

Examples
$root = bisectionsolve("e^x-x^2", -3, 0)  // ~ -0.703467

discretenewtons

discretenewtons(function, xmin, xmax, [guess])

Applies a discrete form of Newton's method to find a root of the function on the interval [xmin, xmax]. Starts at the midpoint of the interval, or at guess if provided. If the method fails to converge, or converges outside the interval, the method returns null. Make sure the function uses the variable x.

Examples
$root = discretenewtons("x^3-x-2", 1, 2) // 1.521379
$root2 = discretenewtons("x^3-x-2", 1, 2, 1.5)  // starting from guess of 1.5
