forminterval

forminterval(min, max, minmarkertype, maxmarkertype)

Forms an interval string, like "[2,3)".

  • min, max: min and max of the interval.
  • minmarkertype, maxmarkertype: 0 for open, 1 for closed.
Examples
forminterval(2, 3, 1, 0)  // "[2,3)"

linegraph

linegraph(intervals, [xmin, xmax, xscl, width, height])

Creates a line graph.

  • intervals: string or array of strings in interval notation, e.g. "[2,5)".
  • xscl: can be a number, or "labelscl:tickscl" to have labels and tick marks at different spacing.
Examples
linegraph("[2,5)")
linegraph(array("(-oo,-1]", "[2,5)"), -10, 10)

linegraphbrackets

linegraphbrackets(intervals, [xmin, xmax, xscl, width, height])

Creates a line graph using brackets/parens instead of open/closed dots.

  • intervals: string or array of strings in interval notation, e.g. "[2,5)".
  • xscl: can be a number, or "labelscl:tickscl" to have labels and tick marks at different spacing.
Examples
linegraphbrackets("[2,5)")
linegraphbrackets("[2,5)", -10, 10, "2:1")  // labels every 2, ticks every 1

intervalstodraw

intervalstodraw(interval, xmin, xmax)

Converts an interval like (-oo,4]U(3,5] into an array of strings suitable for use as $answers for a drawing question with $answerformat = 'numberline'.

  • xmin, xmax: the min and max of the drawing grid, if different than the default -5 to 5.
Examples
$answers = intervalstodraw("(-oo,4]U(3,5]")
$answers = intervalstodraw("[-8,8]", -10, 10)
