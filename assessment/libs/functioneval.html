A smattering of macros for evaluating complicated functions. Version 1.0 Oct 4 2009

  • fe_lpfnc

fe_lpfnc

fe_lpfnc(x, xarray, yarray, [periodic])

Evaluates a linear piecewise function (connect-the-dots) from the provided array of x and y values at the given x value. Requires the x array to be in increasing order.

  • x: the x value at which to evaluate the function.
  • xarray, yarray: arrays of x and y values defining the piecewise-linear function.
  • periodic: Optional - if true, then xarray values define the domain of one period, and if x is outside that domain, f(x) = f(x-p) will be used. The domain is starting x <= x < last x, so make sure the last x = x[0]+p.
Examples
$y = fe_lpfnc(1.5, array(0,1,2,3), array(0,2,1,3))  // 1.5 (halfway between 2 and 1)
$yperiodic = fe_lpfnc(5, array(0,2,4), array(0,3,0), true)  // wraps using the period
