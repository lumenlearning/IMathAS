bio_getcodonname

bio_getcodonname(codon)

Looks up the name of the amino acid that is transcribed from a codon.

Examples
bio_getcodonname("AUG")  // "Methionine"

bio_randcodon

bio_randcodon()

Outputs an array of a random codon. The output looks like: (codon, 3 letter abbreviation, full name).

Examples
list($codon, $abbr, $name) = bio_randcodon()
// e.g. "AUG", "Met", "Methionine"

bio_anticodon

bio_anticodon(string)

Takes in a string of bases, and outputs a matching anticodon string. (A → T, T → A, G → C, C → G, U → A)

Examples
bio_anticodon("AUG")  // "TAC"

bio_translation

bio_translation(string of DNA bases)

Takes in a string of DNA bases, and outputs an RNA string. (A → U, T → A, G → C, C → G)

Examples
bio_translation("TACGGG")  // "AUGCCC"

bio_splitcodons

bio_splitcodons(string)

Splits a string into an array made of sets of 3 codons.

Examples
bio_splitcodons("AUGCCCUAG")  // array("AUG", "CCC", "UAG")
