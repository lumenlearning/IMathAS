Macro Library vector

Vector functions. Version 1.0, April 4, 2006

In these functions, vectors are represented by arrays.

  • dotp
  • crossp
  • vecnorm
  • vecsum
  • vecdiff
  • vecprod
  • veccompareset
  • veccomparesamespan

dotp

dotp(a, b)

Dot product of vectors a and b.

  • a, b: arrays (1+ elements).
Examples
dotp(array(1,2,3), array(4,5,6))  // 32

crossp

crossp(a, b)

Cross product of vectors a and b.

  • a, b: arrays (3 elements). Returns array.
Examples
crossp(array(1,0,0), array(0,1,0))  // array(0,0,1)

vecnorm

vecnorm(a)

Returns the magnitude of vector a.

  • a: array (1+ elements).
Examples
vecnorm(array(3,4))  // 5

vecsum

vecsum(a, b)

Returns the sum a+b.

  • a, b: arrays of the same length.
Examples
vecsum(array(1,2,3), array(4,5,6))  // array(5,7,9)

vecdiff

vecdiff(a, b)

Returns the difference a-b.

  • a, b: arrays of the same length.
Examples
vecdiff(array(4,5,6), array(1,2,3))  // array(3,3,3)

vecprod

vecprod(a, c)

Returns the product c*a.

  • a: array.
  • c: scalar.
Examples
vecprod(array(1,2,3), 3)  // array(3,6,9)

veccompareset

veccompareset(a, b)

a and b are both arrays of vectors, e.g. $a = array(array(1,2,3),array(3,4,5)).

Returns a value between 0 (no overlap) and 1 (sets are equivalent), calculated as n(a intersect b)/max(n(a),n(b)).

Examples
$a = array(array(1,2,3), array(3,4,5))
$b = array(array(1,2,3), array(0,0,0))
veccompareset($a, $b)  // 0.5

veccomparesamespan

veccomparesamespan(A, B)

Determines if span(A) = span(B), where A is a linearly independent set.

  • A, B: arrays of vectors, e.g. $A = array(array(1,2,3),array(3,4,5)). Note that if you want to use the columns of a matrix as A, you'll need to transpose the matrix.

Returns true or false.

Examples
$A = array(array(1,0,0), array(0,1,0))
$B = array(array(2,0,0), array(0,3,0))
veccomparesamespan($A, $B)  // true
