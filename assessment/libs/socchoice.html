apportion

apportion(populations, seats, method)

Returns an array giving the apportionment to each state.

  • populations: array or list of state populations.
  • seats: number of seats to apportion.
  • method: method to use: "hamilton", "jefferson", "adams", "webster", "huntington", "lowndes".
Examples
$seats = apportion(array(150,300,550), 21, "hamilton") // [3,6,12]

apportion_info

apportion_info(populations, seats, method)

  • populations: array or list of state populations.
  • seats: number of seats to apportion.
  • method: method to use: "hamilton", "jefferson", "adams", "webster", "huntington", "lowndes".

Returns an array of:

  • 0) the divisor
  • 1) an array of quotas (decimal values)
  • 2) an array of initial lower quotas (hamilton, jefferson, lowndes); an array of initial upper quotas (adams); an array of initial rounded quotas (webster); an array of initial allocations (huntington)
  • 3) an array of geometric means (huntington); an array of decimal/whole ratios (lowndes); an array of decimal remainders (hamilton); nothing (others)
  • 4) valid modified divisors, in interval notation (jefferson, adams, webster)
Examples
$divisor, $quotas, $initial, $extra, $moddiv = apportion_info(array(150,300,550), 21, "jefferson")
// $divisor = 47.619
// $quotas = [3.15, 6.3, 11.55]
// $initial (lower quotas) = [3,6,11]
// $extra = [] nothing for jeffersons
// $moddiv = "(42.857,45.8333]" divisors that would work

banzhafpower

banzhafpower(weights, quota, [format])

Calculates the Banzhaf power index for the voting system. Returns an array of power indexes.

  • weights: list or array of voter weights.
  • quota: voting system quota.
  • format: defaults to decimal power index values. "counts": power counts, "frac": power index as fraction strings.

Based on http://www.hippasus.com/resources/socialsoftware/banzhaf.py (CC-BY)

Examples
$power = banzhafpower(array(6,3,2), 7) // [.6,.2,.2]
$counts = banzhafpower(array(6,3,2), 7, "counts") // [3,1,1]

shapleyshubikpower

shapleyshubikpower(weights, quota, [format])

Calculates the Shapley-Shubik power index for the voting system. Returns an array of power indexes.

  • weights: list or array of voter weights (max: 7 voters).
  • quota: voting system quota.
  • format: defaults to decimal power index values. "counts": power counts, "frac": power index as fraction strings.
Examples
$power = shapleyshubikpower(array(6,3,2), 7) // ~[.667,.127,.167]
