Normal curve functions

normalcurve

normalcurve(mu, sigma, a, b, [axislabel, x, dirx, y, diry, color, width, height])

Draws a normal curve with mean mu and standard deviation sigma. This function is good for a basic normal curve, vertically scaled to fit, with axis labels at multiples of sigma.

  • a, b: draws on bounds a < x < b, with axis markings at mu+k*sigma.
  • axislabel: gives an axis label.
  • x, dirx, y, diry: optionally can shade a region based on one or two values x and y. Indicate shade direction using dirx and diry, which can be "left" or "right". You can use null for y to set later values.
  • color: indicates color of the shaded regions.
  • width, height: pixels for the displayed graph. Defaults to 400 by 150.
Examples
$img = normalcurve(0, 1, -3, 3) // standard normal from -3 to 3
$img2 = normalcurve(0, 1, -3, 3, "z", -1, "left") // shaded left of -1

normalcurve2

normalcurve2(mu, sigma, a, b, [axisspacing, ymax, axislabel, x, dirx, y, diry, color, width, height])

Draws a normal curve with mean mu and standard deviation sigma. This function allows for axis spacing other than multiples of sigma, and does not scale the graph vertically to fit.

  • a, b: draws on bounds a < x < b.
  • axisspacing: specify axis spacing, or defaults to sigma.
  • ymax: set max of the vertical axis, or "auto" for ymax to autoscale.
  • axislabel: gives an axis label.
  • x, dirx, y, diry: optionally can shade a region based on one or two values x and y. Indicate shade direction using dirx and diry, which can be "left" or "right". You can use null for y to set later values.
  • color: indicates color of the shaded regions.
  • width, height: pixels for the displayed graph. Defaults to 400 by 150.
Examples
$img = normalcurve2(0, 1, -3, 3, 0.5, "auto")

normalcurve3

normalcurve3(mu, sigma, a, b, [axislabel, x, dirx, y, diry, color, q, dirq, r, dirr, color2, width, height])

Draws a normal curve with mean mu and standard deviation sigma. This function is good for a basic normal curve, vertically scaled to fit, with axis labels at multiples of sigma. This version allows for the shading of TWO regions with different colors.

  • a, b: draws on bounds a < x < b, with axis markings at mu+k*sigma.
  • axislabel: gives an axis label.
  • x, dirx, y, diry, color: optionally can shade a region based on one or two values x and y. Indicate shade direction using dirx and diry, which can be "left" or "right".
  • q, dirq, r, dirr, color2: optionally can shade a second region based on one or two values q and r. Indicate shade direction using dirq and dirr, which can be "left" or "right". You can use null for y, q, and/or r to set later values.
  • width, height: pixels for the displayed graph. Defaults to 400 by 150.
Examples
$img = normalcurve3(0, 1, -3, 3, "z", -1, "left", null, null, "blue", -1.5, "left", null, null, "red")
// shades blue left of -1, red left of -1.5
// like for showing critical region and p-value region
