chem_disp

chem_disp(compound)

Formats a compound for display as HTML.

Examples 
$a = chem_disp("H2O")    // "H<sub>2</sub>O"

chem_mathdisp

chem_mathdisp(compound)

Formats a compound for better display in math mode.

Examples 
$a = chem_mathdisp("H2O")    // "`H_2 O`

chem_eqndisp

chem_eqndisp(reactants, products, coefficients, [arrow, phases])

Generates a display string for a chemical reaction.

  • reactants: array of compounds.
  • products: array of compounds.
  • coefficients: array of coefficients for both reactants and products. The output of chem_balancereaction can be used for this.
  • arrow: optional, a string for the arrow in the reaction. Defaults to ->
  • phases: optional, an array of phases for both reactants and products.
Examples 
$coeffs = chem_balancereaction(array("H_2","O_2"), array("H_2 O"))
$eqn = chem_eqndisp(array("H_2","O_2"), array("H_2 O"), $coeffs)
// returns "2 H_2 + O_2 -> 2 H_2 O"

chem_isotopedisp

chem_isotopedisp(element, super, sub, [noitalic])

Formats a math display string for an isotope with the given element, superscript, and subscript. Set noitalic to true to un-italicize the element.

Examples 
$a = chem_isotopedisp("C", 14, 6)  // "{::}_{\ \ 6}^{14}C"

chem_getfamily

chem_getfamily(atomic number)

Returns the family given the atomic number.

Values may be: "", "Noble gas", "Alkaline", "Alkaline Earth", "Halogen", "Transition Metal", "Lanthanide", or "Actinide"

Examples
chem_getfamily(11)  // "Alkaline"

chem_randelementbyfamily

chem_randelementbyfamily(family)

Returns the atomic number of a random element from a family.

Valid families are: "Other", "Noble gas", "Alkaline", "Alkaline Earth", "Halogen", "Transition Metal", "Lanthanide", or "Actinide"

Examples
$el = chem_randelementbyfamily("Halogen")

chem_diffrandelementsbyfamily

chem_diffrandelementsbyfamily(family, n)

Returns an array of n atomic numbers for random elements from a family.

Valid families are: "Other", "Noble gas", "Alkaline", "Alkaline Earth", "Halogen", "Transition Metal", "Lanthanide", or "Actinide"

Examples
$els = chem_diffrandelementsbyfamily("Halogen", 3)

chem_getrandcompound

chem_getrandcompound(type)

Returns an array of (compound name, compound formula).

Valid types are: twobasic, twosub, threeplus, parens. If type is not supplied, compound is chosen randomly from all.

Examples
$name, $formula = chem_getrandcompound("twobasic") // eg "Lithium bromide", "Li Br"

chem_getdiffrandcompounds

chem_getdiffrandcompounds(n, type)

Returns an array of n arrays of (compound name, compound formula).

Valid types are: twobasic, twosub, threeplus, parens. If type is not supplied, compound is chosen randomly from all.

Examples
$compounds = chem_getdiffrandcompounds(3, "twobasic")

chem_decomposecompound

chem_decomposecompound(compound)

Breaks a compound into an array of elements and an array of atom counts.

Examples
$elements, $counts = chem_decomposecompound("H2O")
// $elements = array("H","O"); $counts = array(2,1)

chem_getcompoundmolmass

chem_getcompoundmolmass(compound, [round])

Gets the molecular mass of the given compound.

  • round: decimals to round the individual atoms' molecular mass to during calculations. Default: no additional rounding (4 decimal place accuracy). Special value: .5. Rounds all values to whole numbers, except Cl and Cu to nearest .5.
Examples
chem_getcompoundmolmass("H2O")  // 18.0152

chem_randcation

chem_randcation([group, name type, include uncommon])

  • group: simple, basic (simple + NH_4), polyvalent, or all. Default = all.
  • name type: "common" (iron (II)) or "alternate" (ferrous). Default = common.
  • include uncommon: set true to include francium, radium, cadmium, etc.

Returns array(symbol, charge, name).

Examples
$symbol, $charge, $name = chem_randcation("basic")

chem_randanion

chem_randanion([group, name type, include uncommon])

  • group: simple, polyatomic, or all. Default = all.
  • name type: "common" (bicarbonate) or "alternate" (hydrogen carbonate). Default = common.
  • include uncommon: set true to include selenide, peroxide.

Returns array(symbol, charge, name).

Examples
$symbol, $charge, $name = chem_randanion("polyatomic")

chem_makeioniccompound

chem_makeioniccompound(cation, anion)

Takes a cation and anion - these are the arrays returned by chem_randcation and chem_randanion.

Returns array(formula, name).

Examples
$cation = chem_randcation("basic")
$anion = chem_randanion("simple")
$formula, $name = chem_makeioniccompound($cation, $anion)
// e.g. "Ag Cl", "silver chloride"

chem_getsolubility

chem_getsolubility(cation, anion)

Takes a cation array and anion array (from chem_randcation and chem_randanion) and returns solubility information from https://en.wikipedia.org/wiki/Solubility_chart.

Currently missing data for:

  • Cations: Cd, Fr, Ra, Cr 2+, Mn 3+, Co 3+, Ni 3+, Cu 1+, Au 1+, Sn 4+, Pb 4+, Hg_2
  • Anions: N, P, C, Se, ClO, ClO_2, ClO_3, HCO_3, H_2PO_4, HSO_3, HSO_4, IO_3, MnO_4, NO_2, CrO_4, Cr_2O_7, HPO_4, SO_3, SiO_3, AsO_4, PO_3, O_2

Returns array(stateName, abbreviation).

Examples
$cation = chem_randcation("basic")
$anion = chem_randanion("simple")
$state, $abbr = chem_getsolubility($cation, $anion)

chem_balancereaction

chem_balancereaction(reactants, products)

Balances a chemical reaction, returning an array of coefficients for the compounds.

  • reactants: array of compounds.
  • products: array of compounds.
Examples
$coeffs = chem_balancereaction(array("H2","O2"), array("H2O"))  // array(2,1,2)

chem_showmolecule

chem_showmolecule(string, [width, height, alttext, format])

Draws a 2D molecule diagram.

  • string: The string should be one generated by the molecule helper; just assign the result to something other than $answer and pass it in as the string to this function.
  • width: width (default 300)
  • height: height (default 225)
  • alttext: use to provide an accessible alternative text for the drawing.
  • format: set to "condensed" to show atom labels on drawing.
Examples
$img = chem_showmolecule($molstring, 300, 225, "Structure of ethanol")

chem_vsepr

chem_vsepr(steric number, number of lone pairs)

Looks up molecular geometry based on VSEPR Theory.

Returns an array containing:

  • Molecular Geometry
  • Electron Geometry
  • Bond Angle (approximate for a general case)
  • Polarity (assuming equal electronegativity)
  • Hybridization of central atom
Examples
chem_vsepr(3, 1)  // array('Bent', 'Trigonal Planar', '<120', 'Polar', 'sp^2')
