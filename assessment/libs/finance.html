futureValue

futureValue(interest, bigN, littleN, payment, principal)

Returns the Future Value of an investment or loan based on periodic constant payments and a constant interest rate.

  • interest: annual interest rate as a percent. Example: for 7.5%, $interest should = 7.5
  • bigN: the total number of periods for the investment/loan.
  • littleN: the number of compounding periods per year. Example: for quarterly, $littleN = 4
  • payment: the periodic amount paid into or withdrawn from the account. Should be a negative value if paid into the account. Should be a positive value if withdrawn from the account.
  • principal: the initial value of the account. Should be a negative value if paid into the account. Should be a positive value if withdrawn from the account.

Returns the Future Value of the account. Returns a positive value if the investment/loan resulted in a positive return. Returns a negative value if the investment/loan resulted in an amount owed.

Examples
// future value of $1000 principle, $500 deposits/quarter 
// 10 quarters at 7.5% interest
$fv = futureValue(7.5, 10, 4, -500, -1000)

presentValue

presentValue(interest, bigN, littleN, payment, accruedVal)

Returns the Present Value of an investment or loan based on periodic constant payments and a constant interest rate.

  • interest: annual interest rate as a percent. Example: for 7.5%, $interest should = 7.5
  • bigN: the total number of periods for the investment/loan.
  • littleN: the number of compounding periods per year. Example: for quarterly, $littleN = 4
  • payment: the periodic amount paid into or withdrawn from the account. Should be a negative value if paid into the account. Should be a positive value if withdrawn from the account.
  • accruedVal: the Future Value of the investment. Should be a positive value if the investment/loan resulted in a positive return. Should be a negative value if the investment/loan resulted in an amount owed.

Returns the Present Value of the account. Returns a positive value if the initial principal was withdrawn from the account (Loan). Returns a negative value if the initial principal was invested into the account (Savings).

Examples
// present value of $2000 future value, $500 deposits/month 
// 60 months at 7.5% interest
// $pv = presentValue(7.5, 60, 12, -500, 2000)

payment

payment(interest, bigN, littleN, principal, accruedVal)

Returns the periodic Payment for an investment or loan based on periodic constant payments and a constant interest rate.

  • interest: annual interest rate as a percent. Example: for 7.5%, $interest should = 7.5
  • bigN: the total number of periods for the investment/loan.
  • littleN: the number of compounding periods per year. Example: for quarterly, $littleN = 4
  • principal: the Present Value of the investment. Should be a positive value if the account is a loan. Negative if the account is savings.
  • accruedVal: the Future Value of the investment. Should be a positive value if the investment/loan resulted in a positive return. Should be a negative value if the investment/loan resulted in an amount owed.

Returns the Periodic Payment or Withdrawal for the Loan or Investment. Positive value if withdrawn from the account. Negative if deposited into the account.

Examples
// monthly payments on a $1000 loan
// for 48 months at 5% interest, ending with 0 owed
$pmt = payment(5, 48, 12, 1000, 0)

numberOfPeriods

numberOfPeriods(interest, littleN, payment, principal, accruedVal)

Returns the number of Payments or Withdrawals for an investment or loan based on periodic constant payments and a constant interest rate.

  • interest: annual interest rate as a percent. Example: for 7.5%, $interest should = 7.5
  • littleN: the number of compounding periods per year. Example: for quarterly, $littleN = 4
  • payment: the periodic amount paid into or withdrawn from the account. Should be a negative value if paid into the account. Should be a positive value if withdrawn from the account.
  • principal: the Present Value of the investment. Should be a positive value if the account is a loan. Negative if the account is savings.
  • accruedVal: the Future Value of the investment. Should be a positive value if the investment/loan resulted in a positive return. Should be a negative value if the investment/loan resulted in an amount owed.

Returns the number of Payments or Withdrawals for the Loan or Investment, rounded to two decimal places.

Examples
// how many $10000 annual withdrawals
// can be made from a $100000 account earning 8% interest
$n = numberOfPeriods(8, 1, 10000, -100000, 0)

interest

interest(bigN, littleN, payment, principal, accruedVal)

Returns the Interest Rate of the Investment/Loan as a percentage, based on periodic constant payments.

  • bigN: the total number of periods for the investment/loan.
  • littleN: the number of compounding periods per year. Example: for quarterly, $littleN = 4
  • payment: the periodic amount paid into or withdrawn from the account. Should be a negative value if paid into the account. Should be a positive value if withdrawn from the account.
  • principal: the Present Value of the investment. Should be a positive value if the account is a loan. Negative if the account is savings.
  • accruedVal: the Future Value of the investment. Should be a positive value if the investment/loan resulted in a positive return. Should be a negative value if the investment/loan resulted in an amount owed.

Returns the Interest Rate of the Investment/Loan as a percentage, rounded to two decimal places.

Examples
// what rate will allow 10 quarterly payments to 
// pay off at $1000 loan with $110 quarterly payments
$rate = interest(10, 4, -110, 1000, 0)

vmTVM

TVM()

Returns an iframe for displaying a Time Value of Money Financial Calculator. Provides students with a virtual manipulative they can use to calculate various values of an investment or loan.

Takes no parameters. Returns the embedded iframe code for inserting the applet into an IMathAS question.

Examples
$tvmcalc = TVM()
