chunktext

chunktext(text, size)

Adds spaces between chunks of text of the given size.

Examples
chunktext("HELLOWORLDABC", 5)  // "HELLO WORLD ABC"

randfiveletterword

randfiveletterword()

Returns a random five-letter word.

Examples
$word = randfiveletterword()  // e.g. "HOUSE"

randmilphrase

randmilphrase([removespaces], [pad])

Returns a random military-ish phrase like "at noon march towards target".

  • removespaces: Optional - if set to true, all spaces will be removed.
  • pad: Optional - if set to a number N, then the phrase will be padded with random characters for a transposition cipher with N columns.
Examples
$phrase = randmilphrase()  // e.g. "at noon march towards target"
$phrase2 = randmilphrase(true)  // e.g. "atnoonmarchtowardstarget"
$phrase3 = randmilphrase(true, 6)  // padded to a multiple of 6 characters

shiftcipher

shiftcipher(message, shift, [alpha], [rotate])

Encrypts the message using a basic Caesar shift substitution cipher.

  • message: the text to encrypt.
  • shift: can be a numeric value (e.g., 3 means A encrypts as D), or can be a letter indicating what A encrypts as.
  • alpha: Optional - should be "alpha" for A-Z or "alphanum" for A-Z0-9. Defaults to "alpha".
  • rotate: Optional - if set to true, then the shift will be increased after each letter is encrypted. Can be set to a number to shift by more than 1 after each letter.
Examples
shiftcipher("ATTACK", 3)  // "DWWDFN"
shiftcipher("ATTACK", "D")  // "DWWDFN" (D is what A shifts to)
shiftcipher("ATTACK", 1, "alpha", true)  // rotating shift, e.g. "BVWEHQ"

randsubmap

randsubmap([charset])

Generates a random substitution mapping.

By default uses A-Z. Set charset to "alphanum" for A-Z0-9.

Examples
$map = randsubmap()  // random mapping of A-Z, e.g. "QWERTYUIOPASDFGHJKLZXCVBNM"
$map2 = randsubmap("alphanum")  // random mapping of A-Z0-9

subcipher

subcipher(message, submap, [alpha], [rotate])

Encrypts the message using a general substitution cipher.

  • message: the text to encrypt.
  • submap: a string consisting of all characters of the character set in the desired mapping order.
  • alpha: Optional - should be "alpha" for A-Z or "alphanum" for A-Z0-9. Defaults to "alpha".
  • rotate: Optional - if set to true, then the shift will be increased after each letter is encrypted. Can be set to a number to shift by more than 1 after each letter.
Examples
$map = randsubmap()
$enc = subcipher("ATTACK", $map)  // encrypts ATTACK using the random mapping

transcipher

transcipher(message, columns, [order])

Encrypts a message using a basic transposition cipher, writing the message in rows, each with the given number of columns. The encrypted message is then formed by reading down the columns. The message is padded if needed with A's.

  • message: the text to encrypt.
  • columns: number of columns to write the message into.
  • order: Optional - by default, the columns are read left to right. If an array of numbers (0 to cols-1) is provided, they specify the order to read the columns. If a word is given instead, it serves as the code-word, and columns are read in the alphabetic order of the letters in the word (e.g. "day" would mean 2 1 3, since a is the first of the letters alphabetically, d is 2nd, etc).
Examples
transcipher("ATTACKATDAWN", 3)  // reads columns left to right
transcipher("ATTACKATDAWN", 3, array(2,0,1))  // reads columns in a custom order
transcipher("ATTACKATDAWN", 3, "day")  // columns read in alphabetic order of "day"

modularexponent

modularexponent(base, exponent, modulus)

Calculates base^exponent mod modulus.

Examples
modularexponent(4, 13, 497)  // 445

cryptorsakeys

cryptorsakeys(p, q)

Given two primes, p and q, returns array(n, e, d) where n and e form the public key, and d is the private key.

Examples 
$n, $e, $d = cryptorsakeys(61, 53) // [3233, 7, 1783]
// $n=3233, $e=7 is the public key; $d=1783 is the private key
