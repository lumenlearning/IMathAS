ineqplot

ineqplot("funcstring", [xmin, xmax, ymin, ymax, labels, grid, width, height])

Graphs a set of functions, shading above or below each graph as specified.

  • funcstring: one string or array of strings, each with the format (function and filltype are required):
    function of x,filltype,fillcolor,linecolor,dash,strokewidth
  • filltype: "above" or "below".
Examples
ineqplot("x^2-1,above")
ineqplot(array("x^2-1,above,red", "3-x,below,blue,blue,dash"), -5, 5, -5, 5)

ineqbetweenplot

ineqbetweenplot("funcstring", [xmin, xmax, ymin, ymax, labels, grid, width, height])

Graphs a set of functions, shading the area satisfying all above/below specifications.

  • funcstring: one string or array of strings, each with one of these cx_format2std (function and filltype are required):
    • function of x,above or below,linecolor,dash,strokewidth,fillcolor
    • x=number,right or left,linecolor,dash,strokewidth,fillcolor
Examples
ineqbetweenplot(array("x^2-1,above", "2-x,below"))
ineqbetweenplot(array("x=-1,right", "x^2,above"), -5, 5, -5, 5)
