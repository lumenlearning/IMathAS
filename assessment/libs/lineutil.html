Macro Library lineutil

Various line utility functions

  • lineboundarycoord

lineboundarycoord

lineboundarycoord(xmin, xmax, ymin, ymax, m, b, [order])

Looks for where the line crosses the graph boundary. Supply the graph boundaries, and m and b of the line.

  • xmin, xmax, ymin, ymax: the graph boundaries.
  • m, b: the slope and y-intercept of the line.
  • order: Optional - an array of "right", "top", "left", "bottom" indicating the order of search. Default: ["right","top","left","bottom"]

Returns array(x, y, loc), where loc is a best guess for where it would be good to put the label.

Examples
// find where 2x+1 intersects window
$x, $y, $loc = lineboundarycoord(-10, 10, -10, 10, 2, 1) // [4.5, 10, 'belowright']
// use it to place a label with the graph
$graph = showplot(["2x+1","text,$x,$y,f(x),black,$loc"],-10,10,-10,10,2,2)

// change order to check sides
$x, $y, $loc = lineboundarycoord(-10, 10, -10, 10, 2, 1, array("left","bottom","right","top"))
