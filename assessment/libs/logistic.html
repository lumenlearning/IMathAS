logisticregression

logisticregression(x_array, y_array)

Fits a logistic curve to the data given in x_array and y_array. Returns an array(a,b,c), which are the coefficients for the logistic curve c / (1 + a e^(-b x)). The values will be accurate to 4 or 5 significant digits, but likely will not exactly match regression performed by other technology.

Examples
$x = array(0,1,2,3,4,5)
$y = array(1,2,4,7,9,9.7)
$coeffs = logisticregression($x, $y)  // ~[12.48, 1.07, 10.45]

logisticpredict

logisticpredict(coefficients, x)

Uses an array of coefficients, returned by logisticregression, to predict the output for the given x value.

Examples
$coeffs = logisticregression($x, $y)
$predicted = logisticpredict($coeffs, 6) // ~10.197

logisticsolve

logisticsolve(coefficients, y)

Uses an array of coefficients, returned by logisticregression, to solve for an input given a y value.

Examples
$coeffs = logisticregression($x, $y)
$xval = logisticsolve($coeffs, 8)  // ~3.487
