baseconvert

baseconvert(num, from, to)

Converts the number num from base from to base to.

If a base is >10, then each place should be given (or will be returned) separated by commas, e.g. 5,12,3.

Examples
baseconvert(23, 5, 7)  // 16. converts 23-base5 to base7
baseconvert("5,12,3", 16, 10)  // 1475. converts a base-16 number with places 5,12,3 to base10

hexdec

hexdec(a)

Converts a hexadecimal string to a decimal number.

Examples
hexdec("1A")  // 26

dechex

dechex(a)

Converts a decimal number to a hexadecimal string.

Examples
dechex(26)  // "1A"

asciitodec

asciitodec(char)

Returns the ASCII decimal code for a character.

Examples
asciitodec("A")  // 65

dectoascii

dectoascii(num)

Returns the ASCII character for a decimal value.

Examples
dectoascii(65)  // "A"

draw_mayan_number

draw_mayan_number(number, [width, mode])

Draws a representation of the given base-10 number as a Mayan number.

  • number: the base-10 number to represent.
  • width: the width of the image (default 50). The height is auto-adjusted based on the number of positions needed.
  • mode: default "standard" for pure base-20 system. Set to "modified" for the "long count" modified vigesimal system, where the second position can only be up to 17 and the third position represents multiples of 18*20.
Examples
$img = draw_mayan_number(429)
$img2 = draw_mayan_number(429, 80, "modified")

draw_babylonian_number

draw_babylonian_number(number, [height, drawzero])

Draws a representation of the given base-10 number as a Babylonian number.

  • number: the base-10 number to represent.
  • height: the height of the image (default 40). The width is auto-adjusted based on the number of positions needed.
  • drawzero: default false (so zero positions are left blank). Set to 1 to draw angled wedges for zeros in middle positions only, 2 to draw for zeros in all positions.
Examples
$img = draw_babylonian_number(3661)
$img2 = draw_babylonian_number(3661, 60, 1)
