Macro Library calculus

A collection of numerical calculus routines. These are intended to do numerical calculations where they are specifically needed, not substitute for entering symbolic derivatives or integrals.

Version 1.0 Oct 19, 2009

  • calculusdiffquotient
  • calculusnumint

calculusdiffquotient

calculusdiffquotient(function, variable, a, h)

Calculates the difference quotient (f(a+h)-f(a))/h.

Examples 
calculusdiffquotient("x^2", "x", 3, 0.1)  // 6.1

calculusnumint

calculusnumint(func, var, a, b, n, method)

Does a numerical integral approximation on the interval [a, b] with n subdivisions using the specified method.

  • method: one of "left", "right", "midpoint", "trapezoidal", or "simpsons".
Examples
calculusnumint("x^3", "x", 0, 1, 4, "simpsons")   // .25
