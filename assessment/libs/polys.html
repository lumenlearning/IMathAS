formpoly

formpoly(coefficients, powers or degree)

Creates a polynomial object. Use writepoly to create a display form of the polynomial.

  • coefficients: list/array of coefficients.
  • powers or degree: list/array of powers, or highest degree. Highest degree assumes coefficients correspond to consecutively decreasing powers.
Examples
// 3x^2+1
$p = formpoly([3,1], [2,0])  // coefficients and powers
$p2 = formpoly([3,0,1], 2)  // same, w/ coeffs for all powers and max degree

formpolyfromroots

formpolyfromroots(stretch, roots, [multiplicities])

Create a polynomial object from roots. Use writepoly to create a display form of the polynomial.

  • stretch: a stretch factor; the A in A*(x-root)(x-root)...
  • roots: an array of the roots (zeros, x-intercepts). For a complex root pair a±bi provide a single array [a,b] as the root.
  • multiplicities (optional): an array of multiplicities of the roots; assumed to be all 1 if not provided.
Examples
$p = formpolyfromroots(2, array(1,3))  // 2(x-1)(x-3)
$p2 = formpolyfromroots(1, array(0,2), array(1,2))  // x(x-2)^2

writepoly

writepoly(poly, [var, showzeros])

Creates a display form for a polynomial object.

  • poly: polynomial object, created with formpoly.
  • var: input variable. Defaults to x.
  • showzeros: optional, defaults to false. If true, shows zero coefficients.
Examples
$p = formpoly(array(3,1), array(2,0))
$out = writepoly($p)  // "3x^2+1"
$out = writepoly($p, "t", true)  // "3t^2+0t+1"

writepolyfrac

writepolyfrac(poly, [var, showzeros])

Creates a display form for a polynomial object, converting decimal coefficients to fractions where possible.

  • poly: polynomial object, created with formpoly.
  • var: input variable. Defaults to x.
  • showzeros: optional, defaults to false. If true, shows zero coefficients.
Examples
$p = formpoly(array(0.5,0.25), array(2,0))
$out = writepolyfrac($p)  // "1/2x^2+1/4"

addpolys

addpolys(poly1, poly2)

Adds two polynomials, arranging terms from highest to lowest powers.

  • poly1: first polynomial.
  • poly2: second polynomial.
Examples
$sum = addpolys($p1, $p2)

subtpolys

subtpolys(poly1, poly2)

Subtracts polynomials (poly1 - poly2), arranging terms from highest to lowest powers.

  • poly1: minuend polynomial.
  • poly2: subtrahend polynomial.
Examples
$diff = subtpolys($p1, $p2)

multpolys

multpolys(poly1, poly2)

Multiplies two polynomials.

  • poly1: first polynomial.
  • poly2: second polynomial.
Examples
$prod = multpolys($p1, $p2)

divpolys

divpolys(poly1, poly2)

Divides polynomials (poly1 / poly2).

  • Returns an array with two elements: quotient and remainder, each a polynomial object.
  • poly1: dividend polynomial.
  • poly2: divisor polynomial.
Examples
$quotient, $remainder = divpolys($p1, $p2)

scalepoly

scalepoly(poly, c)

Multiplies each term of poly by constant c.

  • poly: polynomial to scale.
  • c: constant multiplier.
Examples
$scaled = scalepoly($p, 3)

derivepoly

derivepoly(poly)

Computes the derivative of poly.

  • poly: polynomial to differentiate.
Examples
$deriv = derivepoly($p)

polypower

polypower(poly, power)

Calculates polypower.

  • poly: base polynomial.
  • power: exponent (integer).
Examples
$squared = polypower($p, 2)

quadroot

quadroot(a, b, c)

Solves the quadratic equation ax^2 + bx + c = 0. Returns an array of the two solutions ordered smaller then larger. If no real solutions exist, an array of "DNE" strings is returned.

  • a, b, c: coefficients of the quadratic.
Examples
quadroot(1, -3, 2)  // array(1, 2)
quadroot(1, 0, 1)  // array("DNE", "DNE")

getcoef

getcoef(poly, degree)

Gets the coefficient corresponding to the specified degree. If no such term is defined, 0 is returned.

  • poly: polynomial object, created with formpoly.
  • degree: degree of the term to get the coefficient of.
Examples
$p = formpoly(array(3,1), array(2,0))
getcoef($p, 2)  // 3
getcoef($p, 1)  // 0

polys_getdegree

polys_getdegree(poly)

Returns the degree of the polynomial.

  • poly: polynomial object.
Examples
$p = formpoly(array(3,1), array(2,0))
polys_getdegree($p)  // 2

describepoly

describepoly(eqn, xints, [invar, outvar, other])

Generates a text description of a polynomial that you can use as an input to replacealttext when the auto-generated table of values isn't sufficient for accessibility. Note the function only describes behavior at roots for multiplicity 1 and 2 behaviors.

  • eqn: string equation, like "3(x-2)(x-3)"
  • xints: array of x-intercepts, like [2,3]
  • invar: (optional) input variable (default "x")
  • outvar: (optional) output variable (default "y")
  • other: (optional) array of other features you want to note each of form [xval, 'description'] or 'description' ex: [[3, 'local minimum'], 'graph is concave up']. if xval is given, it will calculate the y and generate "description at (xval,yval)"
Examples
$desc = describepoly("3(x-2)(x-3)", array(2,3))
$desc2 = describepoly("(x-1)^2", array(1), "x", "y", array([1, 'local minimum']))

describerational

describerational(eqn, xints, vas, ha, [invar, outvar, other])

Generates a text description of a rational function that you can use as an input to replacealttext when the auto-generated table of values isn't sufficient for accessibility. Note the function only describes behavior at roots and asymptotes for multiplicity 1 and 2 behaviors.

  • eqn: string equation, like "2((x-3)(x-2))/((x-4)(x+1))"
  • xints: array of x-intercepts, like [2,3]
  • vas: array of x-coordinates of vertical asymptotes, like [-1,4]
  • ha: horizontal asymptote value, null if none, ex: 2
  • invar: (optional) input variable (default "x")
  • outvar: (optional) output variable (default "y")
  • other: (optional) array of other features you want to note each of form [xval, 'description'] or 'description' ex: [[3, 'local minimum'], 'slant asymptote of y=x+1']. if xval is given, it will calculate the y and generate "description at (xval,yval)"
Examples
$desc = describerational("2((x-3)(x-2))/((x-4)(x+1))", array(2,3), array(-1,4), 2)
