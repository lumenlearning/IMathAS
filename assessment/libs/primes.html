Macro Library primes

Prime numbers, Version 1.0 Feb 11, 2009

  • getprime
  • getprimes
  • isprime

getprime

getprime([digits])

Returns a prime number. If digits is specified, returns a prime with that many digits. 1 to 6 is supported.

Examples
$p = getprime()  // a random prime number, e.g. 2341
$p2 = getprime(3)  // a random 3-digit prime, e.g. 613

getprimes

getprimes(n, [digits])

Returns n different prime numbers. If digits is specified, returns primes with that many digits. 1 to 6 is supported.

Examples
$plist = getprimes(5)  // array of 5 different random primes
$plist2 = getprimes(3, 2)  // array of 3 different random 2-digit primes, e.g. array(13, 47, 89)

isprime

isprime(n)

Returns true or false, indicating if n is prime. Only works for whole numbers less than 1,000,000.

If you need a prime number, use getprime instead.

Examples
isprime(17)  // true
isprime(20)  // false
