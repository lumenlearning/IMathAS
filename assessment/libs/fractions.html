fractionrand

fractionrand(denom)

Returns a proper reduced fraction with the given denominator.

Examples
fractionrand(12)  // e.g. "5/12"

fractiondiffdrands

fractiondiffdrands(min, max, n, [order])

Returns n fractions with different denominators from min to max. Can set order to 'inc' or 'dec' to sort the values.

Examples
fractiondiffdrands(2, 10, 3)  // e.g. array("1/7", "3/4", "5/9")
fractiondiffdrands(2, 10, 3, "inc")  // sorted increasing

fractiondiffdrandsfrom

fractiondiffdrandsfrom(list, n, [order])

Returns n fractions with different denominators from the list. Can set order to 'inc' or 'dec' to sort the values.

Examples
fractiondiffdrandsfrom(array(2,4,6,8), 2)  // e.g. array("1/4", "3/8")

fractionparse

fractionparse(fraction)

Converts a fraction into array(num, denom).

Examples
fractionparse("3/4")  // array(3, 4)

fractiontomixed

fractiontomixed(f)

Converts a fraction string to a mixed number string.

Examples
fractiontomixed("7/2")  // "3 1/2"

fractiontodecimal

fractiontodecimal(f)

Converts a fraction string to a decimal value.

Examples
fractiontodecimal("3/4")  // 0.75

fractionreduce

fractionreduce(f)

Takes a fraction string or array(num,denom). Returns a reduced fraction string.

Examples
fractionreduce("4/8")  // "1/2"
fractionreduce(array(4,8))  // "1/2"

fractionadd

fractionadd(f1, f2, [f3, f4, ...])

Adds a set of fractions, returning a reduced fraction sum.

Examples
fractionadd("1/2", "1/3")  // "5/6"

fractionsubtract

fractionsubtract(f1, f2, [f3, f4, ...])

Subtracts a set of fractions, returning a reduced fraction difference: f1 - f2 - f3 - f4 etc.

Examples
fractionsubtract("5/6", "1/3")  // "1/2"

fractionmultiply

fractionmultiply(f1, f2, [f3, f4, ...])

Multiplies a set of fractions, returning a reduced fraction.

Examples
fractionmultiply("2/3", "3/4")  // "1/2"

fractiondivide

fractiondivide(f1, f2, [f3, f4, ...])

Divides a set of fractions, returning a reduced fraction: f1 ÷ f2 ÷ f3 ÷ f4.

Examples
fractiondivide("1/2", "3/4")  // "2/3"

fractionneg

fractionneg(frac)

Changes the sign of a fraction.

Examples
fractionneg("3/4")  // "-3/4"

fractionpower

fractionpower(frac, power)

Raises a fraction to a power.

Examples
fractionpower("2/3", 2)  // "4/9"

fractionroot

fractionroot(frac, [root, rationalize])

Finds the root of the fraction (defaults to square root). Rationalizes the denominator by default. The output is a string.

Examples
fractionroot("4/9")  // "2/3"
fractionroot("1/2", 2, false)  // not rationalized, e.g. "1/sqrt(2)"
